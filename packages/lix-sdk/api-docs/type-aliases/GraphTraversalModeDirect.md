Direct mode: Only the specified node is included.

No parent or child traversal is performed.

```mermaid

    A[ChangeSet A]
    B[ChangeSet B]
    C[ChangeSet C]
    B --> A
    C --> B
    click A "Selected (direct)"
```

Selected node: A
Included: only A
